Stock Plan Manager — Windows

Installation guide. About 2 minutes.

1. Install the app

  1. Download StockPlan-Setup.exe from the releases page.
  2. Double-click StockPlan-Setup.exe in your Downloads folder.
SmartScreen warning on first launch: because Stock Plan Manager isn't signed yet, Windows will show "Windows protected your PC". Click More info, then Run anyway. This is one-time per installer download.
  1. Step through the installer wizard: Next → Next → Install.
  2. If prompted by Windows, allow administrator access — this is needed to install into Program Files.
  3. The installer silently installs the Microsoft Visual C++ Runtime if your system doesn't have it. This is normal and takes about 10 seconds.
  4. When the wizard finishes with Launch Stock Plan Manager checked, click Finish. Your browser opens automatically at http://localhost:4002 within 20 seconds.

2. Using the app

  1. The app runs in the background — there is no Stock Plan Manager window. You interact with it entirely through your browser tab.
  2. To stop the server: click the red Quit button in the top-right of any page in the app, then close the browser tab.
  3. To start the server again later: open the Start Menu, type Stock Plan Manager, click the app. The browser tab opens automatically.

3. Where your data lives

Stock Plan Manager stores everything on your local machine. Specifically:

  • %USERPROFILE%\.stock_plan\stock_plan.db — your portfolio database (SQLite)
  • %USERPROFILE%\.stock_plan\profile.json — your display name

Nothing leaves your PC. The only external network calls are to public stock-price and FX-rate sources (Yahoo Finance, RBI reference rates).

4. Troubleshooting

Browser tab won't load after install

The server takes 5–20 seconds to start on first launch (runs initial database setup). Refresh the page. If it's still failing after a minute, open Task Manager → Details, look for any erl.exe / beam.smp processes — if none, re-launch from the Start Menu.

"Windows protected your PC" with no "Run anyway" button

You may have very strict SmartScreen settings. Right-click StockPlan-Setup.exe → Properties → bottom of the General tab, tick Unblock → OK. Re-run the installer.

App can't find the database after a Windows account change

If you signed in with a different Windows user, the app installed for all users but the data lives in your user profile. Sign back in to the account where you originally created data, or copy .stock_plan\ between user profiles.

5. Uninstall

  1. Stop the server: open the app, click Quit.
  2. Open Settings → Apps → Installed apps, find Stock Plan Manager, click ⋯ → Uninstall.
  3. (Optional) remove your data: in File Explorer go to %USERPROFILE%, delete the .stock_plan folder. This deletes the database, profile, and any uploaded files.
