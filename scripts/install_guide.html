Stock Plan Manager

Quick install for macOS — about 2 minutes.

1. Install the app

  1. The Finder window for the disk image should already be open. If not, double-click StockPlan.dmg in your Downloads folder.
  2. Drag the StockPlan icon onto the Applications shortcut in that window.
  3. Eject the disk image: press ⌘E in Finder, or click the ⏏ next to "StockPlan" in the Finder sidebar.

2. First launch — allow the app

Because Stock Plan Manager isn't signed by an Apple-registered developer, macOS will block the first launch. This is one-time per Mac.

  1. Open Applications in Finder. Double-click StockPlan.
  2. macOS will show: "StockPlan can't be opened because Apple cannot check it for malicious software." Click Done.
  3. Open System Settings (Apple menu → System Settings).
  4. Click Privacy & Security in the sidebar.
  5. Scroll down to the Security section. You'll see: "StockPlan was blocked to protect your Mac." Click Open Anyway.
  6. Authenticate with Touch ID or your password.
  7. Go back to Applications and double-click StockPlan again.
  8. You'll see a new dialog with an Open button. Click it.
Important: Steps 1 and 2 must happen in that order. The "Open Anyway" button only appears in System Settings after you've attempted to launch the app once and gotten the block message.

After this one-time setup, double-clicking StockPlan launches it normally — no further prompts.

3. Using the app

  1. The first launch opens your browser at http://localhost:4002 automatically.
  2. The app has no Dock icon — it runs in the background and you interact with it through the browser tab.
  3. To stop the server: click the red Quit button in the top-right of any page in the app, then close the browser tab.

4. Troubleshooting

"StockPlan can't be opened" with no entry in Privacy & Security

This happens if the DMG was downloaded through a sandboxed app like WhatsApp, Messages, or Mail. macOS blocks anything downloaded by those apps from being executed, with no override path. Workaround: download the DMG using Chrome, Firefox, or directly from the GitHub release page in any browser, then re-install.

App icon is blank in Finder

macOS caches icons. After installing a new version, open Terminal and run:

touch /Applications/StockPlan.app && killall Finder && killall Dock

Browser tab won't load after launching the app

The server takes 5–10 seconds to start on first launch (it runs database migrations). Refresh the page. If still failing after 30 seconds, the BEAM process may have crashed — open Activity Monitor, search for beam.smp, kill it, and try again.

5. Uninstall

  1. Stop the server if it's running (Quit button in the app, or kill beam.smp in Activity Monitor).
  2. Drag StockPlan from Applications to the Trash.
  3. Optional — remove your data: in Terminal, run rm -rf ~/.stock_plan. This deletes your portfolio database, profile, and uploaded files.
